# eretz

## E**[retz](https://github.com/BlackBo6/eretz)**

A small application that send emails

## To run the application

### Run the backend ( port 4000 )

```bash
git clone https://github.com/BlackBo6/eretz.git
cd eretz/backend
npm install
npm run dev
```

### Run the frontend ( port 3000 )

```bash
cd eretz/frontend
npm install
npm run start
```
